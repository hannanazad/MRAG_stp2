# MUTCD-150 runtime assets

This directory is safe to load in the model-execution environment. It contains the fixed question-only benchmark, the resumable runner, and a default model registry.

It intentionally excludes the gold answers and evaluator metadata.

## Immutable question file

- File: `mutcd_benchmark_questions_v1.jsonl`
- Records: 150
- SHA-256: `3a04b1d620a80704eefac34c565449a0cb8814e781dd6d73b8afb77318b954b2`

Do not edit this file. A changed hash creates a different benchmark.

## Editable model registry

`model_registry.json` is the repository default. The notebook copies it to Google Drive on first use so model selections can be changed persistently without modifying the benchmark or committing every experimental edit.

## Runtime storage

The notebook reads the runner and question set from the cloned GitHub repository and writes run outputs under `MyDrive/MRAG/benchmark_runs/` through `CFG.base_dir`.
